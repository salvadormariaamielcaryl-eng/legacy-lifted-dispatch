# Legacy Lifted Dispatch Website

A responsive static website based on the black-and-gold Legacy Lifted design.

## Files

- `index.html` — page content
- `styles.css` — layout, colors, and responsive styling
- `script.js` — mobile navigation and footer year
- `assets/legacy-lifted-logo.jpg` — logo file

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and the `assets` folder.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/root`, then click **Save**.
6. GitHub will provide the public website link.

## Contact form

The contact form sends inquiries to:

`teamdispatch.legacylifted@gmail.com`

It uses FormSubmit and works with GitHub Pages. After the first test submission, open the activation email from FormSubmit and confirm it once.

## Before publishing

- Replace the sample phone number if needed.
- Add real Facebook and Instagram links.
- Update the services and careers content as needed.
- Add a custom domain in GitHub Pages settings if available.
